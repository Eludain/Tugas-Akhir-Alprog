# Gamifikasi Sistem Layanan Perpustakaan
Output dari proyek ini adalah aplikasi konsol C++ yang memodernisasi sistem peminjaman buku perpustakaan konvensional. Tidak hanya mencatat transaksi, sistem ini menerapkan konsep **Gamifikasi** untuk meningkatkan keterlibatan anggota melalui sistem Poin dan Papan Peringkat (*Leaderboard*).

# Fitur yang Dimiliki:
1. Fitur Tampilkan Daftar Buku seperti ID, Nama, Stok Buku, dan Jumlah Buku yang Sedang Dipinjam.
2. Fitur Peminjaman. pengguna memasukan Nama dan ID Buku yang ingin dipinjam.
3. Fitur Pengembalian, pengguna mengembalikan buku yang dipinjam
4. Sistem Poin dan Papan Peringkat, pengguna mendapatkan poin setiap melakukan peminjaman dan pengembalian buku
